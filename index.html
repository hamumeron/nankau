DevHub セットアップ

GitHub と Cloudflare の API トークンを登録すると、
リポジトリ・Pages を一括管理できます。

⬡ GitHub Personal Access Token GitHub

リポジトリの作成・閲覧・コミット履歴の取得に使用します。
スコープは repo が必要です。

1
GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic) を開く
2
「Generate new token (classic)」をクリック。Note に名前を入力し、repo スコープにチェック
3
「Generate token」→ 表示されたトークン（ghp_...）をコピーして下に貼り付け
⚠ ブラウザの localStorage に保存されます。共有PC では使用しないでください。

▲ Cloudflare API Token Cloudflare

Pages プロジェクトの一覧・作成・デプロイ履歴の取得に使用します。
権限は Cloudflare Pages:Edit が必要です。

1
Cloudflare Dashboard → My Profile → API Tokens を開く
2
「Create Token」→「Custom token」→ Permissions に Account / Cloudflare Pages / Edit を追加
3
「Continue to summary」→「Create Token」→ 表示されたトークンをコピーして下に貼り付け
4
Dashboard の URL 末尾 /{account_id}/... から Account ID も取得
Cloudflare Dashboard URL: dash.cloudflare.com/{この部分}/pages
